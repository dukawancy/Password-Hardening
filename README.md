# Password Hardening 🛡️

Password Hardening is a sleek, premium, real-time password strength evaluator designed with modern cybersecurity UX principles. It provides users with instantaneous, interactive feedback on password complexity based on five structural criteria to help them harden their credentials.

## 🚀 Features

* **Real-time Analysis:** Evaluates passwords dynamically as the user types—no page refreshes or submit buttons required.
* **Visual Complexity Indicator:** Features a responsive progress bar that scales and changes color (`Weak` 🔴, `Medium` 🟡, `Strong` 🟢) depending on how many security criteria are met.
* **Security Checklist:** Tracks 5 essential password hardening factors:
    * Minimum of 8 characters
    * Uppercase letters
    * Lowercase letters
    * Numeric digits
    * Special characters
* **Mask/Unmask Toggle:** Built-in visibility switch for user convenience.
* **Modern Premium UI:** Built using a glassy, responsive layout with smooth CSS micro-interactions, custom SVG icons, and a dark mode aesthetic tailored for tech-forward tools.
* **Zero Dependencies:** Crafted completely with vanilla HTML5, CSS3, and modern JavaScript.
